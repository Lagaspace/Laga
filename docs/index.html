Laga library

Laga is a Genetic Algorithm (GAs) Library written in C#. The library includes a series of classes Laga.GeneticAlgorithm to design and build different GAs architectures: population > evaluate > select > Crossover > mutation > replace population. Laga also includes classes for mathematic operations Laga.Numbers, Classes to import and export data Laga.IO, Display basic information Laga.Graphics, geometry operations through Laga.Geometry and list operations and different useful tools through the class in Laga.Tools.

The library has been designed to be simple use and engage more designers into multi-objective optimization problems. Laga allows anyone to try different genetic operators until find the best architecture to the problem to solve and is open source.
